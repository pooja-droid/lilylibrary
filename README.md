# Features

**User Authentication**
- Custom hashing algorithm 
- Secure login and registration system  

**Book Management**
- Browse and search books by title, author, genre
- Borrow, renew, reserve, and return books  
- View detailed descriptions and availability status

**Reviews & Ratings**
- Users can leave reviews and ratings on each book
- Display of average ratings and review history

**Book Recommendation System**
- Uses cosine similarity with user borrowing history and reviews
- Built using NumPy, Pandas, and scikit-learn
- Suggests similar books to each user based on past preferences
